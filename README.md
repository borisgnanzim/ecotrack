
# Ecotrack

## Organisation du projet

Le projet **Ecotrack** est organisé de manière à regrouper le **backend** et le **frontend** dans une même arborescence pour faciliter la gestion et le déploiement.

### Détails :
- **backend/** : Contient la logique serveur, les API et les services.
    - **services/users/** : Module dédié à la gestion des utilisateurs (CRUD, authentification, etc.).
- **frontend/** : Contient l'application cliente (ex. VueJS, React, etc.) pour l'interface utilisateur.
